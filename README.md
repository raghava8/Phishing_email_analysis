

Creating Scripts for phishing email Analysis

url_ext.py:  
It is used to extract URLs from the phishing emails  
This script works for both (pasting the email content and uploading .eml file)


url_ext_app.py:  
This script will give you an User Interface to upload the .eml file and a Text box to paste the email content  
To use this script, install "streamlit" by using the command "pip install streamlit beautifulsoup4"  
To run this script, use the command "python -m streamlit run url_ext_app.py"
